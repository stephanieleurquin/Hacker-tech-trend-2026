# Hacker-tech-trend-2026

Analyse et modélisation du dataset **Hacker News Tech Trend Velocity**.

## 📊 Description

Ce projet explore les posts sur Hacker News avec l’objectif de :

- Analyser la viralité des posts (`Is_Viral`)  
- Visualiser les relations entre score, commentaires et vélocité  
- Construire un modèle de classification pour prédire si un post devient viral  

---

## 🧰 Technologies utilisées

- **Python 3**  
- **Jupyter Notebook**  
- **Pandas** (manipulation des données)  
- **NumPy** (calculs numériques)  
- **Scikit-learn** (modélisation ML)  
- **Matplotlib / Seaborn** (visualisation)  

---

## 🚀 Contenu du projet

- `Notebook.ipynb` : Notebook principal avec nettoyage, analyse et visualisations  
- `data/` : Dossier contenant le dataset (optionnel si trop gros)  
- `README.md` : Ce fichier
